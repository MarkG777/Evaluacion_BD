  DEFINICIÓN DE VARIABLE OBJETIVO (TARGET)
=================================================================

  Target: alerta_temperatura
  Definición: temperatura > 35.0°C
  Base científica: Owens (1971) y Seeley (1985) —
    temperatura interior de colmena saludable: 32–36°C.
    Por encima de 35°C se activan las 'abejas de ventilación'
    (fanners), lo que correlaciona con cambios en sonido y humedad.

  Distribución del target:
    Clase 0 (sin alerta) :   8364 (97.56%)
    Clase 1 (alerta)     :    209 (2.44%)

  MÉTODO 3: Chi-Cuadrado (χ²) — Variables Categóricas
=================================================================

  El test de Chi-Cuadrado evalúa si existe una asociación
  estadísticamente significativa entre cada variable discreta/
  categórica y el target (alerta_temperatura).

  H₀: la variable es independiente del target.
  Si p-valor < α (0.05) → rechazamos H₀ → la variable es relevante.

  Requiere valores no negativos → se aplica MinMaxScaler previo.

  Resultados Test χ² (α = 0.05):

  es_noche              : χ²=     98.2280  p=0.00000000  ✅ Significativo
  tipo_sensor_enc       : χ²=     71.5408  p=0.00000000  ✅ Significativo
  periodo_dia_enc       : χ²=     16.5792  p=0.00004666  ✅ Significativo
  lluvia                : χ²=      0.9715  p=0.32431762  ❌ No significativo
  es_fin_semana         : χ²=      0.3314  p=0.56485730  ❌ No significativo

  → Descartadas por χ² no significativo (p≥0.05): ['lluvia', 'es_fin_semana']
